## 근거
환경측정기기의 형식승인·정도검사 등에 관한 고시 **별표(성능시험 기준 — 먹는물 분야)**

## 위치
CL은 **먹는물 분야** 기기 — 수질TMS(물환경보전법)와 별개 체계 적용

---

## 성능시험 항목 및 기준

CL 성능시험 항목은 정도검사와 동일한 항목 + 추가 항목으로 구성.

### 정도검사와 공통 항목

| 항목 | 기준 |
|------|------|
| 반복성 | 측정범위의 **2.0% 이하** |
| 제로드리프트 | 측정범위의 **3.0% 이하** |
| 스팬드리프트 | 측정범위의 **3.0% 이하** |
| 직선성 | 주입농도값의 **±5.0% 이하** |
| 응답시간(T90) | **10분(90%) 이하** |

### 성능시험 추가 항목 (형식승인 전용)

| 항목 | 내용 |
|------|------|
| 절연저항 | 전기 절연 성능 확인 |
| 내전압 | 규정 전압 인가 시 이상 없음 |
| 종합성능시험 | **7일 연속 운전** 중 성능 유지 확인 |

---

## 정도검사 vs 성능시험 차이

| 구분 | 정도검사 | 성능시험(형식승인) |
|------|---------|----------------|
| 대상 | 사업장 설치 기기 | 출시 전 모델 |
| 주체 | KTL 등 지정기관 | KTL 등 지정기관 |
| 직선성 측정점 | 1점 (M 1점) | 다점 측정 |
| 종합성능시험 | 없음 | 7일 연속 |
| 전기 안전 시험 | 없음 | 있음 |
| 목적 | 현장 기기 성능 확인 | 모델 설계 기준 인증 |

---

## 표준용액 (DPD법 기준)
- 시험방법 코드: **ES 04353** / ES 04309.3
- 측정 원리: DPD(N,N-diethyl-p-phenylenediamine)법
- 제로용액: 잔류염소 불검출 정제수
- 스팬용액: 차아염소산나트륨(NaOCl) 희석액 (농도 기지)
