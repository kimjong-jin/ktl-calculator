# 표준용액 제조 일반 원칙

tags: [표준용액, 제조, 교정, 스팬용액, 제로용액, 정도검사]
links: [[정도검사-계산방법]], [[법령-수질오염공정시험기준]], [[표준용액-TOC]], [[표준용액-TN]], [[표준용액-TP]], [[표준용액-SS]], [[표준용액-pH]], [[표준용액-DO]], [[표준용액-COD]], [[표준용액-BOD]], [[표준용액-CL]], [[표준용액-TU-포르마진]]

## 개요
정도검사 및 일상 교정에 사용하는 표준용액의 공통 원칙.
각 항목별 상세 제조법은 개별 노드 참조.

---

## 용액 종류

| 종류 | 정의 | 목적 |
|------|------|------|
| 제로용액 (Zero) | 측정 대상 물질 농도 = 0 (또는 최소) | 제로점 교정, 반복성 측정 |
| 스팬용액 (Span) | 측정범위 내 특정 농도 | 스팬점 교정, 반복성·직선성 측정 |
| 직선성용액 | M1·M2·M3 (저·중·고농도) | 직선성 시험 |

---

## 스팬용액 농도 선정 기준
```
스팬용액 농도 ≥ 측정범위 최대값 × 85%
```
- 예: 측정범위 0~100 mg/L → 스팬 85 mg/L 이상
- TMS 정도검사 기준 — 고시 별표2의1

| 항목 | 표준물질 | 근거 고시 |
|------|---------|---------|
| TOC | 프탈산수소칼륨 (KHP) | ES 04313 |
| TN | 질산칼륨 (KNO₃) | ES 04363 |
| TP | 인산이수소칼륨 (KH₂PO₄) | ES 04362 |
| SS | 카올린 현탁액 (여과중량법 실측) | ES 04303 |
| COD | 프탈산수소칼륨 (KHP, ThOD 환산) | ES 04315 |
| BOD | 포도당+글루타민산 (GGA) | ES 04305 |

## 공통 주의사항

1. **초순수 사용**: TOC ≤ 0.1 mg/L, 탁도 ≤ 0.02 NTU
2. **유리기구 세척**: 산 세척(HCl) 후 초순수 헹굼
3. **건조 후 칭량**: 흡습성 물질은 105°C 건조 후 데시케이터 냉각
4. **냉장 보관**: 대부분 4°C 이하 갈색병 보관
5. **유효기간 준수**: 제조일 기록, 통상 1개월
6. **농도 검증**: 인증표준물질(CRM) 또는 교차검증으로 확인 권장
