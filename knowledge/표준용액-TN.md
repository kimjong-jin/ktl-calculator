# TN (총질소) 표준용액 제조

## 근거
수질오염공정시험기준 **ES 04363** — 산화환원법(자외선흡광법)

---

## 표준물질: 질산칼륨 (KNO₃)

| 항목 | 값 |
|------|-----|
| 분자식 | KNO₃ |
| 분자량 | 101.10 |
| 질소(N) 함량 | 14 / 101.10 × 100 = **13.85%** |

---

## 농도별 칭량량 계산식
```
필요 KNO₃(mg/L) = 목표 TN 농도(mg/L) ÷ 0.1385
```

### 주요 농도별 칭량량 (1L 기준)

| 목표 TN (mg/L) | KNO₃ 칭량량 (mg) |
|--------------|----------------|
| 1 | 7.2 |
| 5 | 36.1 |
| 10 | 72.2 |
| 20 | 144.4 |
| 50 | 360.9 |
| 100 | 721.8 |
| 500 | 3,608.9 |
| 1000 | 7,217.8 |

---

## 조제 순서 (1L 기준)

1. KNO₃를 **105°C에서 2시간 건조**, 데시케이터 냉각
2. 정밀 칭량 후 초순수 약 900 mL에 용해
3. 1,000 mL 메스플라스크에 정용
4. 냉장 보관

---

## 주의사항
- 초순수 TN ≤ 0.1 mg/L 확인 후 사용
- 유효기간: **냉장 1개월**
- 고농도 원액(1000 mg/L) 제조 후 희석 사용 권장
- 암모니아성 질소 계열은 **염화암모늄(NH₄Cl)** 사용 가능
  - NH₄Cl MW: 53.49, N 함량: 14/53.49 × 100 = 26.17%
  - 1000 mg/L TN → NH₄Cl 3,819 mg/L
