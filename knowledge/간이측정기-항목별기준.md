# 간이측정기 성능인증 항목별 기준

## 근거
환경분야 시험·검사 등에 관한 법률 시행규칙 **별표 2의3** (간이측정기 성능인증 등급 판정 기준)

> 간이측정기는 형식승인 기기보다 **기준이 완화**됨 (현장 신속 측정 목적).
> 등급은 **1등급** 또는 **등급 외** 2단계만 존재.

---

## 1. 대기 분야

| 항목 | NO₂ | CO | O₃ |
|------|-----|----|----|
| **반복성** (실내) | ≤ 10.0% | ≤ 10.0% | ≤ 10.0% |
| **직선성** (실내) | ≤ 15.0% | ≤ 15.0% | ≤ 15.0% |
| **상대정확도** (실외) | ≤ 30% | ≤ 30% | ≤ 30% |

→ 하나라도 초과 시 **등급 외**
