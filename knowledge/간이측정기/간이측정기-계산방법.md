## 근거
환경분야 시험·검사 등에 관한 법률 시행규칙 **별표 2의3** (간이측정기 성능인증 등급 판정 기준)

---

## 판정 원칙

```
실내시험 항목(반복성·직선성·온도보상 등) 모두 1등급 기준 이내
    AND
실외시험 항목(상대정확도) 1등급 기준 이내
    → 1등급 판정
```

**비고:**
- 평가항목 중 **하나라도 등급 외** → 전체 등급 외
- 실내시험에서 등급 외 → 실외시험 생략, 즉시 등급 외 판정 가능

## 계산 공식

### 반복성
```
RSD(%) = 표준편차(σ) / 평균(x̄) × 100
```

### 직선성
```
직선성 오차(%) = |측정값 - 기준값| / 기준값 × 100
```

### 상대정확도 (실외)
```
상대정확도(%) = |간이측정값 - 주시험법값| / 주시험법값 × 100
```
→ 정도검사 현장적용계수·성능시험 상대정확도와 동일 공식

### 온도보상 (DO 전용)
```
온도보상 오차(%) = |보상값 - 이론값| / 이론값 × 100
```
