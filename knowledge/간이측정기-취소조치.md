## 근거
환경분야 시험·검사 등에 관한 법률 **제9조의3③** / 시행규칙 **제6조의6④**

---

## 등급 외 판정 시 조치

### 실내시험에서 등급 외

```
실내시험 결과 확인
    ↓
반복성·직선성·온도보상 중 하나라도 기준 초과
    ↓
실외시험 생략 가능 → 즉시 등급 외 판정
    ↓
성능인증서 발급 (등급 외 명시)
    ↓
기기 개선 후 재신청 (수수료 재납부)
```

### 실외시험에서 등급 외

```
실내시험 합격
    ↓
실외시험 (상대정확도)
    ↓
상대정확도 기준 초과 → 등급 외 판정
    ↓
성능인증서 발급 (등급 외 명시)
    ↓
측정원리·알고리즘 개선 후 재신청
```

## 등급 외 기기 유통·판매 가능 여부

| 상황 | 허용 여부 |
|------|---------|
| 등급 외 판정 기기 유통 | **가능** (성능인증 자체는 의무) |
| 등급 외 측정결과 공개 | 가능하나 **등급 외 명시 의무** |
| 등급 외 기기를 행정목적 사용 | **불가** (형식승인 기기와 달리 간이측정기는 행정목적 사용 불가) |

---

## 성능인증 취소 사유 (법 제9조의3③)

국립환경과학원장이 성능인증을 취소할 수 있는 경우:

1. **거짓·부정한 방법**으로 성능인증을 받은 경우
2. 인증 후 기기가 **성능인증 기준에 미달**하게 된 경우

### 취소 절차

```
취소 사유 발생
    ↓
국립환경과학원장 취소 결정
    ↓
해당 기기 제조·수입자에게 통지
    ↓
관보 또는 인터넷 홈페이지 공고
```

---

## 재인증 신청

등급 외 판정 또는 취소 후 재신청 시:
- 수수료 전액 재납부
- 개선된 기기로 실내시험부터 재실시
- 이전 성능인증 번호 소멸 → 새 번호 부여

---

## 형식승인 취소·정도검사 부적합과의 차이

| 구분 | 간이측정기 취소 | 형식승인 취소 | 정도검사 부적합 |
|------|-------------|------------|--------------|
| 대상 | 간이측정기 모델 | 형식승인 기기 모델 | 사업장 설치 기기 |
| 즉시 효과 | 공고 후 인증 무효 | 해당 모델 출하 불가 | 측정값 행정목적 불가 |
| 사용자 영향 | 측정결과 공개 제한 | 정도검사 불가 | 즉시 사용 중단 |
| 행정처분 | 없음 | 있음 | 있음 |
