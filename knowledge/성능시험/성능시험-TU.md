---

## 성능시험 항목 및 기준

TU는 **먹는물 분야** — 수질TMS보다 엄격한 기준 적용.

| 항목 | 기준 | 비고 |
|------|------|------|
| 반복성 | 측정범위의 **2.0% 이하** | RSD (수질TMS 3.0%보다 엄격) |
| 제로드리프트 (24h) | 측정범위의 **3.0% 이하** | (수질TMS 5.0%보다 엄격) |
| 스팬드리프트 (24h) | 측정범위의 **3.0% 이하** | |
| 직선성 | **±5.0% 이하** | **1점 측정** (M 1점) |
| 응답시간 (T90) | **10분(90%) 이하** | |
| 절연저항 | **5 MΩ 이상** (AC 500V) | 성능시험 전용 |
| 내전압 | AC 1,500V 1분 이상 없음 | 성능시험 전용 |
| 종합성능시험 | 7일 연속 운전 전 항목 기준 만족 | 성능시험 전용 |

> TU는 현장적용계수 시험 없음 (먹는물 분야)
> 직선성 측정점: **M 1점** (수질TMS 3점과 다름)

---

## 표준용액

| 용액 | 내용 |
|------|------|
| 제로용액 | 0.2 μm 막여과 정제수 (≈ 0.02 NTU) |
| 스팬용액 | 포르마진 현탁액 (측정범위 85% 이상) |
| 직선성용액 | 스팬 농도 1점 (M = 측정범위의 45%) |

→ 상세 제조: [[표준용액-TU-포르마진]]

---

## 직선성 기준값
```
M = 측정범위 × 0.9 / 2 = 측정범위의 45%
예: 측정범위 0~10 NTU → M = 4.5 NTU
```

## 수질TMS vs 먹는물 기준 비교

| 구분 | 수질TMS | 먹는물(TU) |
|------|---------|----------|
| 반복성 | 3.0% | **2.0%** |
| 드리프트 | 5.0% | **3.0%** |
| 직선성 | ±5.0% (3점) | ±5.0% (1점) |
| 현장적용계수 | 있음 | 없음 |
