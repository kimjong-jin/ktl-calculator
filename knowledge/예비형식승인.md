# 예비형식승인 (Preliminary Type Approval)

tags: [예비형식승인, 신기술, 성능인증, 형식승인]
links: [[용어-관계도]], [[형식승인-절차]], [[법령-환경측정기기고시]]

## 정의
기존 구조·성능 기준을 적용하기 어려운 **새로운 측정방식·원리의 기기**를 위한 별도 형식승인 제도.

## 도입 배경
기존 형식승인 기준은 확립된 측정방식(연소산화, 전기화학 등) 기준으로 설계됨.
UV 흡광, 형광, 레이저 산란 등 신기술 측정방식은 기존 기준 적용 불가
→ 새로운 기준을 만들어 승인하는 예비형식승인 제도 도입

## 절차 (고시 제4조의2~제4조의3)

```
업체
  │
  ▼ 1단계: 사전 적합성 검토 신청 (KTL)
KTL — 5인 이상 전문가 회의 개최
  │ 예비형식승인기준(안) 작성
  ▼
  │ 2단계: 기준 확정 신청 (국립환경과학원)
국립환경과학원 — 20일 이내 심의
  │ 예비형식승인기준 확정
  ▼
  │ 3단계: 예비성능시험 (KTL)
KTL — 확정된 기준으로 예비성능시험 실시
  │ 예비성능시험성적서 발급
  ▼
  │ 4단계: 예비형식승인 신청 (국립환경과학원)
국립환경과학원 — 예비형식승인서 발급
```

## 일반 형식승인과 차이

| 구분 | 일반 형식승인 | 예비형식승인 |
|------|------------|------------|
| 대상 | 기존 기준 적용 가능한 기기 | 신기술 측정방식 기기 |
| 기준 | 고시 별표1 (기존 기준) | 전문가 회의로 신규 기준 마련 |
| 절차 | 성능시험 → 형식승인 | 기준 수립 → 예비성능시험 → 예비형식승인 |
| 소요기간 | 비교적 짧음 | 기준 수립 기간 추가 (수개월) |

## KTL 역할
- 예비형식승인기준(안) 작성 (전문가 회의 주관)
- 예비성능시험 실시 및 성적서 발급
- 단, 자체 개발 기기는 직접 수행 불가 (이해충돌 방지)

## 업계 용어 "성능인증"과의 관계
업계에서 "성능인증"이라 부르는 것이 맥락에 따라:
1. **형식승인** (기존 기준 기기)
2. **예비형식승인** (신기술 기기)
3. **정도검사** (설치 후 현장 검증)
를 의미할 수 있어 맥락 파악 필요.
