# 교정 vs 정도검사 차이

tags: [교정, 정도검사, 비교, 용어]
links: [[용어-관계도]], [[정도검사-개요]], [[정도관리]]

## 자주 혼동하는 두 개념

### 교정 (Calibration)
- **정의**: 측정기기의 지시값과 표준값의 관계를 확인하고 조정하는 행위
- **목적**: 측정기기가 올바른 값을 표시하도록 조정
- **주체**: 사업장 자체 또는 교정기관
- **법적 근거**: 국가표준기본법 (교정 의무는 일부 기기에만 적용)
- **주기**: 수시 (제조사 권고에 따라 일일~월간)
- **법정 의무**: 일반적으로 자율 (일부 기기 제외)

**수질TMS에서의 교정:**
- 제로 교정: 제로용액(0 농도)으로 제로점 조정
- 스팬 교정: 표준용액으로 스팬점 조정
- 주 1회 이상 자동 교정 권장

### 정도검사 (Accuracy Inspection)
- **정의**: 형식승인된 측정기기가 현장에서도 성능 기준을 유지하는지 공인기관이 확인
- **목적**: 법적 유효성 확인 (행정목적 사용 자격 부여)
- **주체**: 환경부 지정 검사기관 (KTL 등)
- **법적 근거**: 환경분야 시험·검사 등에 관한 법률 제11조
- **주기**: 1~2년 (항목별 상이)
- **법정 의무**: 필수 (미수검 시 측정값 행정목적 사용 불가)

## 핵심 차이 요약

| 항목 | 교정 | 정도검사 |
|------|------|---------|
| 성격 | 기술적 조정 행위 | 법정 인증 절차 |
| 수행 주체 | 사업장 or 교정기관 | 지정 검사기관만 가능 |
| 결과물 | 교정 성적서 (일부) | 정도검사 기록부 + 증명서 |
| 법적 효력 | 없음 (일부 제외) | 있음 (행정 자격 부여) |
| 주기 | 수시 | 1~2년 |
| 비용 | 소규모 | 항목당 40만~85만원 |

## 관계
```
교정 (일상 관리) ──→ 정도검사 준비
     ↓
정도검사 전 충분한 교정 → 합격 가능성 높아짐
     ↓
정도검사 합격 → 측정값 법적 유효
```

## 교정이 정도검사를 대체할 수 없는 이유
환경분야 시험검사법 제11조 단서: 국가교정업무 전담기관의 **교정**을 받은 경우 정도검사 면제 가능.
→ 단, 일반 교정기관의 교정은 해당 없음. 국가표준기본법 제14조③ 지정기관만 인정.
