# 성능시험 계산방법 (형식승인용)

tags: [성능시험, 계산방법, 반복성, 드리프트, 직선성, 절연저항, 내전압, 종합성능]
links: [[성능시험-개요]], [[정도검사-계산방법]], [[법령-환경측정기기고시]]

## 개요
성능시험 계산 항목은 정도검사와 공통 항목 + 성능시험 전용 항목으로 구성.
→ 공통 항목(반복성·드리프트·직선성·응답시간·현장적용계수)은 [[정도검사-계산방법]]과 동일한 공식.

---

## 공통 항목 (정도검사와 동일)

### 반복성 (Repeatability)
```
RSD(%) = 표준편차(σ) / 평균(x̄) × 100
```
- 제로·스팬용액 각 3회 이상 측정 → 큰 값으로 판정
- DO: mg/L 절대값, pH: pH 절대값 적용

### 제로·스팬 드리프트 (Zero/Span Drift)
```
드리프트(%) = |24시간 후 측정값 - 초기 측정값| / 측정범위 × 100
```
- 24시간 연속 운전 후 측정
- DO·pH: 측정범위 대신 절대값(mg/L, pH) 기준

### 직선성 (Linearity)
```
직선성 오차(%) = (측정값 - 기준값) / 기준값 × 100
기준값 = 측정범위 × 0.9 / 2 = 측정범위의 45%
```
- 수질TMS: M1(15%)·M2(45%)·M3(90%) 3점 측정 → 최대 오차로 판정
- 먹는물(TU·CL): M 1점만 측정

### 응답시간 T90
```
T90 = 측정값이 스텝 변화의 90%에 도달하는 시간 (단위: 분 또는 초)
```

### 현장적용계수
```
현장적용계수(%) = |Ai - Ci| / Ai × 100
```
- 수질TMS 항목에만 해당 (DO·pH·TU·CL 제외)
- 파생모델: 현장적용계수 시험 제외

## 성능시험 전용 항목

### 절연저항 (Insulation Resistance)
```
절연저항(MΩ) = 인가전압(V) / 측정전류(A)
```

**시험 방법:**
1. AC 500V 인가
2. 절연저항계로 측정
3. 전원 단자 ↔ 외함(접지) 간 측정

**합격 기준:** **5 MΩ 이상**

### 내전압 (Dielectric Withstand Voltage)
**시험 방법:**
1. AC **1,500V** 인가
2. **1분간** 유지
3. 절연파괴·아크·섬락 발생 여부 확인

**합격 기준:** 이상 없음 (절연파괴·아크·섬락 없음)

### 종합성능시험 (7일 연속)
**시험 방법:**
1. 기기를 실제 사용 조건에 가까운 환경에서 **7일(168시간) 연속 운전**
2. 매 24시간마다 반복성·드리프트·직선성 측정
3. 전 기간 동안 모든 항목이 합격 기준 내에 있어야 함

**판정:**
- 7일 중 1회라도 기준 초과 → **부적합**
- 7일 연속 전 항목 기준 이내 → **적합**

**목적:** 장기 안정성, 드리프트 누적, 부품 열화 검증

## 성능시험 vs 정도검사 계산 차이 요약

| 항목 | 정도검사 | 성능시험 |
|------|---------|---------|
| 반복성 | 동일 | 동일 |
| 드리프트 | 동일 | 동일 |
| 직선성 | 동일 | 동일 |
| 응답시간 | 동일 | 동일 |
| 현장적용계수 | 있음 | 있음 (파생모델 제외) |
| 절연저항 | **없음** | **있음** (5 MΩ 이상) |
| 내전압 | **없음** | **있음** (AC 1,500V) |
| 종합성능시험 | **없음** | **있음** (7일 연속) |
