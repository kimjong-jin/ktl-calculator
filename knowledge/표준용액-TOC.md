# TOC 표준용액 제조 (스팬용액)

tags: [표준용액, TOC, KHP, 프탈산수소칼륨, 스팬용액, 조제]
links: [[정도검사-계산방법]], [[법령-수질오염공정시험기준]], [[표준용액-제조일반]]

## 근거
수질오염공정시험기준 **ES 04313** — 연소산화-비분산적외선법

## 사용 시약
**프탈산수소칼륨 (KHP, Potassium Hydrogen Phthalate)**
- 분자식: C₈H₅KO₄
- 분자량: 204.22
- 탄소 함량: (8 × 12) / 204.22 × 100 = **47.06%**

## 농도별 칭량량 계산식
```
필요 KHP(mg/L) = 목표 TOC 농도(mg/L) ÷ 0.4706
```

### 주요 농도별 칭량량 (1L 기준)

| 목표 TOC (mg/L) | KHP 칭량량 (mg) |
|----------------|----------------|
| 10 | 21.2 |
| 20 | 42.5 |
| 50 | 106.3 |
| 90 | 191.2 |
| 100 | 212.5 |
| 200 | 425.1 |
| 500 | 1,062.6 |
| 1000 | 2,125.2 |

## 조제 순서 (1L 기준)

1. KHP를 **105℃에서 2시간 건조** (흡습성 제거)
2. 데시케이터에서 냉각 후 정밀 칭량
3. 초순수 약 900 mL에 완전 용해
4. 1,000 mL 메스플라스크에 정용
5. **무기탄소 제거**: 염산(HCl) 소량 첨가 후 질소 스파징 10분
6. 갈색병 밀봉 냉장 보관

## 주의사항
- 초순수 TOC ≤ 0.1 mg/L 확인 후 사용
- 유효기간: **냉장 1개월** 이내
- 스팬용액 농도 = 측정범위 최대값의 **85% 이상** 권장
  - 예: 측정범위 0~100 mg/L → 스팬 85 mg/L 이상
- 고농도 원액(1000 mg/L) 제조 후 희석하는 방식도 가능

## 제로용액
- TOC 제로용액: TOC ≤ 0.1 mg/L 초순수 (별도 처리 불필요)
