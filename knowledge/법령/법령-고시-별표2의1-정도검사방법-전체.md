> 환경측정기기의 형식승인·정도검사 등에 관한 고시 별표2의1 (2026-01-01 시행)
> 수질(용존산소·화학적산소요구량·생물화학적산소요구량·총질소·총인·총유기탄소·수소이온농도·부유물질) 연속자동측정기 정도검사 방법

---

## 1. 반복성 시험

### 시험 방법
1. 제로용액으로 **3회 이상** 반복 측정 → 표준편차(σ_Z), 평균(x̄_Z) 계산 → RSD_Z(%)
2. 스팬용액으로 **3회 이상** 반복 측정 → 표준편차(σ_S), 평균(x̄_S) 계산 → RSD_S(%)
3. 판정: **RSD_Z와 RSD_S 중 큰 값**으로 합격 여부 결정

### 계산식
```
RSD(%) = 표준편차(σ) / 평균(x̄) × 100
```

### 합격 기준
- TOC·TN·TP·SS·COD·BOD: 측정범위의 **3.0% 이하**
- DO: **0.30 mg/L 이하** (절대값, % 아님)
- pH: **pH 0.10 이하** (절대값)
- TU·CL (먹는물): **2.0% 이하**

---

## 2. 드리프트 시험 (Drift)

### 시험 방법
1. 시험 시작 시 제로용액 측정 2회(Z1, Z2), 스팬용액 측정 2회(S1, S2)
2. **4시간 대기** (먹는물 TU·CL: 2시간)
3. 시험 종료 시 제로용액 측정 2회(Z3, Z4), 스팬용액 측정 2회(S3, S4)

### 계산식
```
제로드리프트(%) = |평균(Z3,Z4) - 평균(Z1,Z2)| / 측정범위 × 100
스팬드리프트(%) = |평균(S3,S4) - 평균(S1,S2)| / 측정범위 × 100
```

### 합격 기준
- TOC·TN·TP·SS·COD·BOD: 제로·스팬 각각 **5.0% 이하**
- DO: 제로드리프트 **0.20 mg/L 이하** / 스팬드리프트 **0.30 mg/L 이하**
- pH: 제로·스팬 각각 **pH 0.10 이하**
- TU·CL (먹는물): 제로·스팬 각각 **3.0% 이하**

---

## 3. 직선성 시험 (Linearity)

### 시험 방법
1. 기준값 계산: `기준값 = 측정범위 × 0.9 / 2 = 측정범위의 45%`
2. M1(저농도), M2(중농도), M3(고농도) 3점 측정
3. 각 측정점에서 오차율 계산
4. 판정: 3점 중 **최대 오차율**로 합격 여부 결정
5. TU·CL: M 1점만 측정

### 계산식
```
직선성 오차(%) = (측정값 - 기준값) / 기준값 × 100
```

### 합격 기준
- TOC·TN·TP·SS·COD·BOD·TU·CL: 주입농도값의 **±5.0% 이내**
- DO: **±0.50 mg/L 이내**
- pH: **pH ±0.10 이내**

---

## 4. 현장적용계수 시험

### 시험 방법
1. 현장 배출수 채취 → 수분석(주시험법)으로 분석 농도(Ai) 2회 측정
2. 동시에 TMS 자동측정값(Ci) 2회 기록
3. 위 과정 2회 반복 (총 2세트)

### 계산식
```
현장적용계수(%) = |수분석 평균(Ai) - TMS 평균(Ci)| / 수분석 평균(Ai) × 100
```

### 합격 기준
| 항목 | % 기준 | 절대값 기준 | 비고 |
|------|--------|-----------|------|
| TOC | 15.0% | 0.45 mg/L | 둘 중 하나 만족 |
| TN | 15.0% | 1.50 mg/L | 둘 중 하나 만족 |
| TP | 15.0% | 0.060 mg/L | 둘 중 하나 만족 |
| SS | 20.0% | 1.0 mg/L | 둘 중 하나 만족 |
| COD | 15.0% | 3.0 mg/L | 둘 중 하나 만족 |
| BOD | 20.0% | 1.0 mg/L | 둘 중 하나 만족 |
| pH | — | **±0.20** (절대값) | pH 단위 |
| DO | 해당 없음 | — | DO는 현장적용계수 없음 |
| TU·CL | 해당 없음 | — | 먹는물은 현장적용계수 없음 |

### 자동 적합 조건
배출수 농도 < **배출허용기준 × 50%** 이면 현장적용계수 시험 자동 적합 처리

### 생략 조건 (물환경보전법 시행령)
수질TMS 관제센터가 정도검사일 기준 **3개월 이내** 상대정확도시험을 실시한 경우 현장적용계수 시험 생략 가능

---

## 5. 응답시간 시험 (T90)

### 시험 방법
1. 안정화 상태에서 시험 시작
2. 스텝 변화 주입 (저농도→고농도 또는 반대)
3. 측정값이 변화량의 90%에 도달하는 시간(T90) 측정

### 합격 기준
| 항목 | T90 기준 |
|------|---------|
| TOC·TN·TP·SS·COD·BOD | **15분 이하** |
| DO | **2분 이하** |
| pH | **30초 이하** |
| TU·CL (먹는물) | **10분(90%) 이하** |

---

## 6. pH 온도보상 시험 (pH 전용)

### 시험 방법
온도 변화 시 온도보상 기능의 정확도 확인.

### 합격 기준
**pH ±0.20 이내**

---

## 검사 순서 및 판정 원칙

```
반복성 → 드리프트 → 직선성 → 현장적용계수 → 응답시간
(pH: 온도보상 추가)

판정: 모든 항목 합격 → 적합
      1개 항목이라도 초과 → 부적합
```
