# 정도검사 — 응답시간 (T90)

tags: [정도검사, 응답시간, T90, 수질TMS, 먹는물, 합격기준]
links: [[정도검사-계산방법]], [[정도검사-항목별기준]], [[정도검사-개요]], [[법령-환경측정기기고시]]

## 정의
측정기기의 측정값이 **스텝 변화(제로→스팬)의 90%에 도달하는 시간**.
T90 = Time to 90% response.

측정 대상: 측정기기가 새로운 농도의 시료를 도입했을 때 얼마나 빠르게 반응하는지.

---

## 항목별 합격 기준

### 수질TMS (물환경보전법)

| 항목 | 기준 | 비고 |
|------|------|------|
| TOC | **15분 이하** | |
| TN | **15분 이하** | |
| TP | **15분 이하** | |
| SS | **15분 이하** | |
| COD | **15분 이하** | |
| BOD | **15분 이하** | |
| DO | **2분 이하** | 전극법 — 가장 빠름 |
| pH | **30초 이하** | 전극법 — 가장 빠름 |

> DO와 pH는 전극 방식으로 반응이 매우 빠름. 나머지(TOC·TN·TP·SS·COD·BOD)는 화학 분석이 필요하여 15분 허용.

### 먹는물 (먹는물수질기준규칙)

| 항목 | 기준 | 비고 |
|------|------|------|
| TU (탁도) | **10분 이하** | 광학 방식 |
| CL (잔류염소) | **10분 이하** | 전기화학·광학 방식 |

---

## 측정 방법 (별표2의1)

1. 제로용액을 기기에 공급하여 측정값 안정화
2. **스팬용액으로 전환** → 시간 측정 시작
3. 측정값이 스팬 농도의 **90%에 도달한 시점** 기록 → 이것이 T90
4. T90 ≤ 기준 → 합격

### T90 계산 예시 (TOC, 스팬 10 mg/L 경우)
- 제로 → 스팬 전환 후 측정값 9.0 mg/L (스팬의 90%) 도달 시간 측정
- 14분 30초 도달 → **합격** (15분 이내)
- 16분 도달 → **부적합** (15분 초과)

---

## 응답시간이 정도검사에서 갖는 의미

응답시간이 길면:
- 배출수 농도 변화에 기기가 즉각 반응 못함
- TMS 실시간 모니터링 기능 저하
- 배출허용기준 초과 상황 탐지 지연

→ 수질 오염 사고 대응 능력과 직결되므로 엄격 관리.

## 응답시간 부적합 시 대처

| 원인 | 조치 |
|------|------|
| 측정셀·배관 내 이전 시료 잔류 | 세척 후 재검사 |
| 시료 공급 유량 부족 | 유량 조정 후 재검사 |
| 검출기 노후·오염 | 검출기 교체/세척 후 재검사 |
| 반응 시약 열화 | 시약 교체 후 재검사 |

---

## 먹는물 TU·CL 응답시간의 특수성

먹는물 기기는 정수장에서 **연속 실시간 모니터링** 목적.
10분 기준은 수질TMS 15분보다 엄격 — 정수 품질 이상 즉각 탐지 필요.

TU·CL 응답시간 시험:
- 제로용액(초순수 또는 무염소수) → 스팬용액 전환
- 측정값이 스팬의 90%에 도달하는 시간 측정
