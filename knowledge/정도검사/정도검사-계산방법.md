# 정도검사 계산방법 (검사 항목별 공식)

tags: [정도검사, 계산, 반복성, 드리프트, 직선성, 현장적용계수]
links: [[정도검사-항목별기준]], [[법령-환경측정기기고시]]

## 반복성 (Repeatability)

### 계산식
```
RSD(%) = 표준편차(σ) / 평균(x̄) × 100
```

### 방법
1. 제로용액(Z) 3회 이상 측정 → RSD_Z 계산
2. 스팬용액(S) 3회 이상 측정 → RSD_S 계산
3. 둘 중 **큰 값**으로 판정

### 합격 기준
- 수질TMS: RSD ≤ 3.0%
- 먹는물(TU·CL): RSD ≤ 2.0%
- DO: ≤ 0.30 mg/L (절대값)
- pH: ≤ pH 0.10 (절대값)

---

## 드리프트 (Drift)

### 제로드리프트 계산식
```
제로드리프트(%) = |평균(Z3,Z4) - 평균(Z1,Z2)| / 측정범위 × 100
```

### 스팬드리프트 계산식
```
스팬드리프트(%) = |평균(S3,S4) - 평균(S1,S2)| / 측정범위 × 100
```

### 측정 순서
- Z1,Z2 (초기 측정) → **2시간(먹는물) 또는 4시간(수질TMS) 대기** → Z3,Z4 (최종 측정)
- S1,S2 (초기 측정) → 동일 대기 → S3,S4 (최종 측정)

### 합격 기준
- 수질TMS: ≤ 5.0% (제로·스팬 각각)
- 먹는물(TU·CL): ≤ 3.0%
- DO: 제로 ≤ 0.20 mg/L, 스팬 ≤ 0.30 mg/L
- pH: ≤ pH 0.10

---

## 직선성 (Linearity)

### 계산식
```
직선성 오차(%) = (측정값 - 기준값) / 기준값 × 100
```

### 기준값
```
기준값 = 측정범위 × 0.9 / 2 = 측정범위의 45%
```

### 방법
- M1(저농도)·M2(중농도)·M3(고농도) 3점 측정
- 각 점의 오차율 중 **최대값**으로 판정
- 먹는물(TU·CL)은 M 1점만 측정 후 판정

### 합격 기준
- 전 항목 공통: ±5.0% 이내 (DO·pH 제외)
- DO: ±0.50 mg/L 이내
- pH: ±0.10 이내

---

## 현장적용계수 (Field Application Factor)

### 계산식
```
현장적용계수(%) = |수분석 평균(Ai) - 현장측정 평균(Ci)| / 수분석 평균(Ai) × 100
```

### 측정 방법
1. 현장에서 배출수 채취하여 수분석(주시험법)으로 측정 → Ai (2회 평균)
2. 동시에 TMS 현장측정값 → Ci (2회 평균)
3. 2회차 반복 실시

### 합격 기준 (항목별 허용오차)
| 항목 | 기준 | 절대값 기준 |
|------|------|-----------|
| TOC | 15.0% | 0.45 mg/L |
| TN | 15.0% | 1.50 mg/L |
| TP | 15.0% | 0.060 mg/L |
| SS | 20.0% | 1.0 mg/L |
| COD | 15.0% | — |
| pH | — | ±0.3 |

### 자동 적합 조건
배출수 농도 < 배출허용기준 × 50% → 자동 적합

### 변동성 큰 시료 처리
비율 기준과 절대값 기준 **모두** 만족해야 함
(예: SS는 20% 이하 AND 1.0 mg/L 이하 동시 만족)

---

## 응답시간 (T90)

### 정의
측정값이 스텝 변화의 90%에 도달하는 시간

### 합격 기준
| 항목 | 기준 |
|------|------|
| TOC | 15분 이하 |
| DO | 2분 이하 |
| pH | 30초 이하 |
| TU·CL (먹는물) | 10분(90%) 이하 |

---

## 최종 판정 원칙
- 모든 검사 항목이 합격 기준 이내 → **적합**
- 1개 항목이라도 기준 초과 → **부적합**
- 현장적용계수 미실시 시 해당 항목 제외하고 판정
