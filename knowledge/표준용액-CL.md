## 스팬용액 — 차아염소산나트륨(NaOCl) 희석액

### 원액 준비
```
시약급 NaOCl 용액 (유효염소 약 5~12% 제품)
→ 유효염소 농도를 요오드적정법으로 정확히 확인
```

### 요오드적정법으로 농도 확인
1. NaOCl 원액 소량 + KI 과량 + HCl 산성화
2. 유리된 I₂를 Na₂S₂O₃ 표준액으로 적정
3. 유효염소 농도(mg/L) 계산

```
유효염소(mg/L) = Na₂S₂O₃ 소비량(mL) × 농도(N) × 35,450 / 시료량(mL)
```

### 희석 방법
```
희석 농도(mg/L) = 원액 농도(mg/L) × V취량(mL) / 전체부피(mL)
```

### 스팬 농도 선정
- 측정범위 최대값의 **85% 이상**
- 예: 측정범위 0~2 mg/L → 스팬 1.7 mg/L 이상

---

## 주의사항
- NaOCl은 **불안정** — 빛·열·pH에 민감, 사용 직전 농도 재확인
- 희석 후 **즉시 사용** (보관 중 농도 감소)
- 갈색병 냉장 보관, 유효기간 **1주일 이내**
- 산성 조건에서 염소가스(Cl₂) 발생 — 환기 필수
