# 정도검사 신청 절차

## 신청 방법
KTL(한국산업기술시험원) 수질TMS 정도검사팀에 직접 연락하여 일정 협의 후 신청.

## 단계별 절차

### 1단계: 사전 일정 협의
- KTL에 연락하여 검사 희망 일정 및 기기 정보(항목, 대수) 협의
- 검사 가능 일정 확인 후 예약

### 2단계: 정도검사 신청서 제출
- 별지 제6호서식 "정도검사 신청서" 작성·제출
- 형식승인서 사본 첨부
- 이전 정도검사 기록부 지참 (재검사 시)

### 3단계: 수수료 납부
- 항목별 수수료 + 출장비 납부 (세금계산서 발행)
- [[수수료]] 참조

### 4단계: 현장 검사 실시
KTL 검사원 사업장 방문 → 다음 순서로 검사:
1. 구조 확인 (형식승인 내용 유지 여부)
2. 반복성 시험
3. 드리프트 시험 (2~4시간 소요)
4. 직선성 시험
5. 현장적용계수 시험 (수분석 필요)
6. 응답시간 시험 (해당 항목)

### 5단계: 정도검사 기록부·증명서 발급
- 적합: 정도검사 기록부 + 정도검사 증명서 발급
- 부적합: 정도검사 기록부만 발급 → [[정도검사-부적합조치]] 참조

## 최초 정도검사 시 유의사항
- 형식승인 후 **사용 전** 반드시 수검
- 수검 없이 측정값 행정목적 사용 시 법령 위반
- 설치 완료 후 안정화 기간(통상 1~2주) 후 신청 권장

## 정도검사 기간 (유효 기간 내 수검 권장)
- 주기 만료 1개월 전 ~ 만료 1개월 후 이내 수검 시 당초 주기 기준으로 다음 주기 산정
- 기간 초과 시 수검일 기준으로 주기 재산정

## 수질TMS 현장적용계수 시험 준비사항
- 수분석 시료 채취 당일 검사 필요
- 배출수 수질이 검사 당일 안정적이어야 함
- 배출허용기준 50% 미만 시 자동 적합 처리 가능
