## 표준물질: 인산이수소칼륨 (KH₂PO₄)

| 항목 | 값 |
|------|-----|
| 분자식 | KH₂PO₄ |
| 분자량 | 136.09 |
| 인(P) 함량 | 30.97 / 136.09 × 100 = **22.76%** |

---

## 농도별 칭량량 계산식
```
필요 KH₂PO₄(mg/L) = 목표 TP 농도(mg/L) ÷ 0.2276
```

## 조제 순서 (1L 기준)

1. KH₂PO₄를 **105°C에서 2시간 건조**, 데시케이터 냉각
2. 정밀 칭량 후 초순수 약 900 mL에 용해
3. 1,000 mL 메스플라스크에 정용
4. 냉장 보관

---

## 주의사항
- 초순수 TP ≤ 0.01 mg/L 확인 후 사용
- 유효기간: **냉장 1개월**
- 고농도 원액(100 mg/L) 제조 후 희석 사용 권장
- 유리기구는 산 세척 후 인산염이 잔류하지 않도록 철저히 헹굼
- 인산염은 유리에 흡착 → 폴리에틸렌(PE) 용기 사용 권장
