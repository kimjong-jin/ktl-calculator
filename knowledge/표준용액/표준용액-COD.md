# COD (화학적산소요구량) 표준용액 제조

## 근거
수질오염공정시험기준 **ES 04315** — 다이크로뮴산칼륨법

---

## 표준물질: 프탈산수소칼륨 (KHP)

| 항목 | 값 |
|------|-----|
| 분자식 | C₈H₅KO₄ (KHC₈H₄O₄) |
| 분자량 | 204.22 |
| 이론 COD (ThOD) | 240 / 204.22 = **1.1752 g O₂/g KHP** |

### 이론 COD 계산 근거
```
2KHC₈H₄O₄ + 15O₂ → 16CO₂ + 6H₂O + K₂O
1 mol KHP (204.22 g) → 7.5 mol O₂ (240 g)
ThOD = 240 / 204.22 = 1.1752 g COD/g KHP
```

---

## 농도별 칭량량 계산식
```
필요 KHP(mg/L) = 목표 COD 농도(mg/L) ÷ 1.1752
```

### 주요 농도별 칭량량 (1L 기준)

| 목표 COD (mg/L) | KHP 칭량량 (mg) |
|----------------|----------------|
| 100 | 85.1 |
| 200 | 170.2 |
| 300 | 255.3 |
| 500 | **425.5** |
| 1000 | 850.9 |
| 2000 | 1,701.9 |

> 가장 많이 사용: **500 mg/L COD → KHP 425.5 mg/L**

---

## 조제 순서 (1L 기준)

1. KHP를 **105°C에서 2시간 건조**, 데시케이터 냉각
2. 정밀 칭량 후 초순수 약 900 mL에 용해
3. 1,000 mL 메스플라스크에 정용
4. 냉장 보관

---

## TOC 표준용액과 공통 사용 가능

KHP는 TOC와 COD **양쪽 표준물질로 동시 사용 가능**:
- TOC 환산: KHP 함량 × 47.06% = TOC 농도
- COD 환산: KHP 함량 × 117.52% = COD 농도
- 예: KHP 425.5 mg/L → TOC 200 mg/L, COD 500 mg/L (동시 확인 가능)

---

## 주의사항
- 유효기간: **냉장 1개월**
- 고농도 원액(1000 mg/L COD) 제조 후 희석 권장
- 다이크로뮴산칼륨(K₂Cr₂O₇) 시약: 발암성 — 취급 주의
