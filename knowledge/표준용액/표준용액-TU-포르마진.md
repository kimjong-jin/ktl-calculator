# 포르마진 탁도 표준용액 제조

tags: [표준용액, TU, 탁도, 포르마진, 황산히드라진, HMTA, 4000NTU]
links: [[정도검사-항목별기준]], [[법령-수질오염공정시험기준]], [[표준용액-제조일반]]

## 근거
먹는물공정시험기준 **ES 05303.1** — 탁도 (탁도계법)
수질오염공정시험기준 **ES 04304**

## 반응 원리
황산히드라진 + 헥사메틸렌테트라민 → **포르마진 폴리머** 생성
→ 빛 산란을 이용한 탁도 표준물질

---

## 필요 시약

### 용액 A — 황산히드라진 용액
```
황산히드라진 (Hydrazine sulfate, N₂H₄·H₂SO₄) 1.000 g
→ 정제수에 용해 → 100 mL 정용
```
농도: 10 g/L

### 용액 B — 헥사메틸렌테트라민 용액
```
헥사메틸렌테트라민 (HMTA, Hexamethylenetetramine, (CH₂)₆N₄) 10.00 g
→ 정제수에 용해 → 100 mL 정용
```
농도: 100 g/L

---

## 농도별 제조 방법

### 공통 반응 절차
1. 용액 A **5.0 mL** + 용액 B **5.0 mL** 혼합
2. **25±1°C 암소**에서 **24시간** 반응 (온도 편차 → 농도 오차)
3. 반응 완료 후 정제수로 정용

### 정용 부피에 따른 농도

| 정용 부피 | 농도 |
|----------|------|
| 10 mL | **4000 NTU** (표준원액) |
| 100 mL | **400 NTU** |
| 1000 mL | **40 NTU** |

---

## 희석 방법 (4000 NTU 원액 기준)

```
희석 농도(NTU) = 4000 × V취한량(mL) / 전체부피(mL)
```

| 목표 농도 | 4000 NTU 취량 | 전체 부피 |
|---------|-------------|---------|
| 400 NTU | 10 mL | 100 mL |
| 200 NTU | 5 mL | 100 mL |
| 100 NTU | 2.5 mL | 100 mL |
| 40 NTU | 1 mL | 100 mL |

---

## 주의사항

- 황산히드라진: **발암성(IARC Group 2B)** — 흄후드 내 취급 필수
- 반응 온도 **25±1°C** 엄수 (온도 벗어나면 농도 오류)
- 완성 원액: **갈색병 냉장 보관**, 유효기간 **1개월**
- 정제수 탁도 ≤ **0.02 NTU** 확인 후 사용
- 제로용액: 0.2 μm 막여과지 통과 정제수 (≈ 0.02 NTU)

## 스팬용액 기준 (TU 정도검사용)
- 스팬용액 농도 = 측정범위 최대값의 **85% 이상**
  - 예: 측정범위 0~10 NTU → 스팬 8.5 NTU 이상
