## 근거
환경분야 시험·검사 등에 관한 법률 시행규칙 **제5조** / **제10조** (형식승인 취소)

---

## 성능시험 결과

### 합격 시
```
성능시험성적서 발급 (KTL)
    ↓
형식승인 신청 (국립환경과학원)
    ↓
형식승인서 발급
    ↓
제품 출시 + 최초 정도검사 수검
```

### 불합격 시
```
성능시험성적서 발급 (부적합 항목 명시)
    ↓
원인 파악 및 기기 개선
    ↓
재시험 신청 (수수료 재납부)
```

---

## 불합격 항목별 주요 원인 및 조치

| 불합격 항목 | 주요 원인 | 조치 |
|-----------|---------|------|
| 반복성 | 측정셀 불안정, 검출기 노이즈 | 측정계통 점검·교체 |
| 드리프트 | 교정 불안정, 온도 민감성 | 온도보상 회로 개선 |
| 직선성 | 검출기 비선형, 변환회로 오차 | 검출기·변환기 교체 |
| 응답시간 | 유로 저항, 반응셀 부피 과다 | 유로 구조 개선 |
| 현장적용계수 | 기준법 대비 측정원리 차이 | 측정원리·알고리즘 개선 |
| 절연저항 | 절연재료 불량, 배선 불량 | 절연 처리 강화 |
| 내전압 | 절연 파괴 | 절연 설계 전면 재검토 |
| 종합성능시험 | 장기 드리프트, 소모품 열화 | 부품 교체 주기 설계 개선 |

---

## 형식승인 취소 사유 (법 제10조)

형식승인 취득 후에도 다음 사유 시 취소 가능:

1. **거짓·부정한 방법**으로 형식승인을 받은 경우
2. **형식승인 기준**에 맞지 않게 된 경우
3. **중요 구조·성능 변경** 후 변경승인을 받지 않은 경우

취소 시:
- 관보·인터넷 게시판 공고
- 기존 출하된 기기는 정도검사 수검 불가 → 사용 중단

## 정도검사 불합격과의 차이

| 구분 | 성능시험 불합격 | 정도검사 부적합 |
|------|--------------|--------------|
| 대상 | 제조사 (모델) | 사업장 (기기) |
| 결과 | 출시 불가 | 측정값 행정목적 사용 불가 |
| 처리 | 기기 개선 후 재시험 | 정비 후 재정도검사 |
| 행정처분 | 없음 (미출시) | 있음 (계속 사용 시) |
